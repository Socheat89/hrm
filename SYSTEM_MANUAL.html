របៀបប្រើ System (Quick Start)

For Admin / HR

  • Login ជា Super Admin ឬ HR → ចូល Dashboard
  • កំណត់ Branches និង Departments
  • ចុះឈ្មោះបុគ្គលិក (Employees) និងកំណត់ Role
  • កំណត់ម៉ោងការងារ (Schedules)
  • Generate Attendance QR (បើប្រើ QR)
  • គ្រប់គ្រង Payroll & Reports

For Employee

  • Login → User Dashboard
  • ចូល /scan ដើម្បី Check-In/Out
  • ស្នើសុំច្បាប់ (Leave Requests)
  • ស្នើសុំថែមម៉ោង (Overtime)
  • មើលប្រាក់ខែ (Payslip)

End-to-End Flow Detail

Authentication & Routing

User ចូល /login → System authenticate → ពិនិត្យ Role → Redirect ទៅ Dashboard តាមតួនាទី។ Route ទាំងអស់ត្រូវបានការពារដោយ Middleware។

Attendance Scan

Scan → System resolve ប្រភេទស្កេន (In/Out) → ពិនិត្យ Status, Leave, Duplicate, GPS, QR → Save to attendance_logs → ជូនដំណឹងតាម Telegram។

Requests (Leave / OT)

Employee submit (Pending) → Admin Review → Approve/Reject + Comment → Update Record & Notify Employee.

Payroll Generation

Admin select period → System loop employees → គណនា Base Salary + OT - Late - Leave → Save Payroll → Allow Employee Download.
